### Prisma Commands
- $ npx prisma
- $ npx prisma db pull (pega estrutura do banco de dados e aplica no schema.prisma)
- $ npx prisma db push (cria estrutura dos models no banco de dados)
- $ npx prisma migrate dev
- $ npx prisma generate
- $ npx prisma db seed
- $ npx prisma studio
- $ npx prisma migrate dev --name initialize
